This group covers hardened images for attack surface mapping, passive subdomain discovery, HTTP probing, and DNS resolution. Each image is built on a minimal Alpine base, compiled from source with `CGO_ENABLED=0` for a fully static binary, pinned to a digest for reproducible pulls, and rebuilt at least weekly to pick up upstream patches.

<Warning>
These tools are designed for authorized security research and penetration testing. Only run them against targets you own or have explicit written permission to test. Unauthorized use may violate computer fraud laws in your jurisdiction.
</Warning>

## Common hardening properties

All images in this group share the following properties:

- **Digest-pinned base images** — Base images are referenced by `sha256` digest, preventing silent tag mutations.
- **Static binaries** — All Go tools are compiled with `CGO_ENABLED=0` and `-trimpath -ldflags="-s -w"` for minimal, reproducible binaries.
- **Minimal final image** — Multi-stage builds copy only the compiled binary into a hardened Alpine base; no build toolchain is present at runtime.
- **Weekly rebuilds** — Images are rebuilt on a schedule so the base OS layer stays up to date with security patches.
